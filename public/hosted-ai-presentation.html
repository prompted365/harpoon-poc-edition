ü¶Ä The Rust Advantage: harpoon-core Performance Primitives

‚ö° Why Rust for Orchestration

The original October PoC was built in Rust for a reason: zero-copy parallelism, predictable latency, and memory safety are critical when coordinating dozens of concurrent GPU requests. The Rust core is compiled to Python (PyO3) and WASM for maximum flexibility.

Core Engine Architecture

// Parallel Fragment Processing with Rayon fn compute_states(&self, fragments: Vec) -> Vec { self.pool.install(move || { fragments .into_par_iter() .with_min_len(self.max_batch) // Dynamic batch sizing .map(FragmentState::new) .collect() }) } // Hygiene Scoring: Code quality metrics for intelligent routing fn hygiene_score(body: &str, language: &str) -> f32 { let token_density = (token_count / line_count / 8.0).clamp(0.0, 1.0); let char_density = (char_count / line_count / 120.0).clamp(0.0, 1.0); let indent_quality = compute_indent_balance(body).clamp(0.0, 1.0); let comment_penalty = (1.0 - comment_ratio / 0.8).clamp(0.0, 1.0); 0.25 * token_density + 0.25 * char_density + 0.25 * indent_quality + 0.15 * comment_penalty + structure_bonus // Language-specific heuristics } // Anchor Hash: SHA3-256 content-addressable task tracking fn compute_anchor_hash(body: &str) -> String { let mut hasher = Sha3_256::new(); hasher.update(body.as_bytes()); hex::encode(hasher.finalize()) }

Envelope Cycle: The Core Orchestration Loop

// Queue-based execution with requeuing for failed tasks fn run_cycle(&self, fragments: Vec, hygiene_threshold: f32) -> HarpoonCycleData { let mut queue: VecDeque = (0..states.len()).collect(); let mut anchors: Vec = Vec::new(); let mut events: Vec = Vec::new(); while let Some(idx) = queue.pop_front() { let score = hygiene_score(&state.body, &state.language); if score >= hygiene_threshold { // Fragment absorbed: task completed successfully anchors.push(state.hash.clone()); events.push(CycleEvent::FragmentAbsorbed { ... }); } else { // Fragment requeued: retry with different strategy queue.push_back(idx); events.push(CycleEvent::FragmentRequeued { ... }); } } HarpoonCycleData { absorbed, pending, events, iterations, anchors } }
